# Sound-Effekte für Quiz-App

Dieses Verzeichnis sollte die folgenden Sound-Dateien enthalten:

## Benötigte Sound-Dateien

1. **correct.mp3** oder **correct.wav** - Sound für richtige Antwort
   - Sollte positiv und ermutigend klingen
   - Empfohlene Länge: 0.5-1 Sekunde
   - Format: MP3 oder WAV (beide werden unterstützt)

2. **incorrect.mp3** oder **incorrect.wav** - Sound für falsche Antwort
   - Sollte neutral oder leicht negativ klingen
   - Empfohlene Länge: 0.5-1 Sekunde
   - Format: MP3 oder WAV (beide werden unterstützt)

3. **new_question.mp3** oder **new_question.wav** - Sound beim Erscheinen einer neuen Frage
   - Sollte neutral oder leicht aufmerksamkeitserregend sein
   - Empfohlene Länge: 0.3-0.8 Sekunden
   - Format: MP3 oder WAV (beide werden unterstützt)

## Schnelle Download-Optionen:

### Option 1: Pixabay (Empfohlen - Keine Registrierung nötig)

1. Gehen Sie zu: https://pixabay.com/sound-effects/
2. Suchen Sie nach:
   - **Für correct**: "success", "correct", "ding", "notification positive"
   - **Für incorrect**: "error", "wrong", "fail", "notification negative"
   - **Für new_question**: "notification", "pop", "click", "ui sound"
3. Laden Sie die Sounds herunter
4. Benennen Sie sie um und platzieren Sie sie hier

### Option 2: Freesound.org (Große Auswahl)

1. Gehen Sie zu: https://freesound.org
2. Erstellen Sie ein kostenloses Konto
3. Suchen Sie nach ähnlichen Begriffen wie oben
4. Laden Sie die Sounds herunter (achten Sie auf die Lizenz!)

### Option 3: Mixkit (Sehr einfach)

1. Gehen Sie zu: https://mixkit.co/free-sound-effects/
2. Suchen Sie nach "notification", "success", "error"
3. Laden Sie die Sounds direkt herunter
4. Benennen Sie sie um und platzieren Sie sie hier

## Download-Anleitung:

1. Laden Sie die Sounds von einer der oben genannten Quellen herunter
2. Benennen Sie die Dateien um in:
   - `correct.mp3` oder `correct.wav`
   - `incorrect.mp3` oder `incorrect.wav`
   - `new_question.mp3` oder `new_question.wav`
3. Platzieren Sie die Dateien in diesem Verzeichnis (`assets/sounds/`)
4. Die App unterstützt automatisch sowohl MP3 als auch WAV-Formate

## Konkrete Suchvorschläge:

### Für correct.mp3/wav:
- "success sound effect"
- "correct answer sound"
- "ding notification"
- "positive notification"

### Für incorrect.mp3/wav:
- "error sound effect"
- "wrong answer sound"
- "fail notification"
- "negative notification"

### Für new_question.mp3/wav:
- "notification sound"
- "pop sound effect"
- "ui click sound"
- "soft notification"

## Hinweis:

Falls keine Sound-Dateien vorhanden sind, verwendet die App automatisch Haptic Feedback (Vibration) als Fallback. Die App funktioniert also auch ohne Sound-Dateien!
