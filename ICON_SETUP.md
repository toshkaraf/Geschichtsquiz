# Настройка иконки приложения

## Быстрая установка

1. **Поместите вашу иконку** в корень проекта как `icon.png` (рекомендуемый размер: 1024x1024 пикселей)

2. **Установите зависимости и сгенерируйте иконки:**
```bash
flutter pub get
flutter pub run flutter_launcher_icons
```

3. **Готово!** Иконка будет автоматически применена для всех платформ (Android, iOS, Web, Windows, macOS)

## Примечание

- Цвет фона адаптивной иконки Android установлен на `#D4A574` (коричневый из вашей иконки)
- Если нужно изменить цвет фона, отредактируйте `adaptive_icon_background` в `pubspec.yaml`
